RAM(ラム)「別名：メモリー」

正式名称 : Random Access Memory (RAM)

読み方:Random Access Memory(ランダム アクセス メモリー)


RAM(ラム)は一時的にデータを保存しておく記憶装置である。

よく説明では、「机」として例えられるが内容は

RAMというのは机に例えるなら

私たちは一時的に「机」を使っています。

使用時間は問いませんが移動の際も机ごと

移動する...なんてことはしませんよね？

それがRAMの仕事なのですがもう一つ

机で作業するときは狭いより広いほうがいろいろ置けて

とても便利ですよね？

パソコンも同じく机は広いほうがいいので

RAMの容量が多いほうが作業しやすいのです。

また机は何個かあったほうがいいとも言えますよね？

なのでパソコンに何枚かRAMを刺すことで

快適に机(RAM)が使用できます。
